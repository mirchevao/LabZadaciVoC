Да се напише програма која од непознат број на цели броеви кои се вчитуваат од СВ ќе го отпечати последно внесениот трицифрен парен број.

#include <stdio.h>
int main()
{
    int a,b;
    
    while(scanf("%d",&a))
    {
        if(((a>99)&&(a<1000))&&(a%2==0))
            b=a;
    }
    printf("%d",b);
    return 0;
}
     Пример влез
500 12 3 5 1 12 32 x
Пример излез
500
