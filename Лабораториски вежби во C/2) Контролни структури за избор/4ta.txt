Да се напише програма во која три цели броеви прочитани од СВ ќе ги подреди по големина во растечки редослед и ќе ги отпечати на екран.

# include <stdio.h>
int main(){
    int a,b,c,d;
    scanf("%d %d %d",&a, &b, &c);
  if (a > b)
          {
              d=a;
              a=b;
              b=d;
          }
  if (a > c)
          {
              d=a;
              a=c;
              c=d;
          } 
  if (b > c)
  {
              d=c;
              c=b;
              b=d;
          }
          printf("%d %d %d", a,b,c);
          return 0;
 }

10 3 2
Пример излез
2 3 10
