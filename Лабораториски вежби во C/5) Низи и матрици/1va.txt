Да се напише програма која за дадена низа А од N (N < 100) реални броеви ќе ја најде и отпечати разликата на
збирот на елементите кои се наоѓаат на парни позиции и збирот на елементите кои се наоѓаат на непарни позиции. 
Броевите од низата и нивниот број N се читаат од СВ. 
Забелешка: Првиот елемент се смета дека е на непарна позиција.

#include <stdio.h>
int main()
{
    int i,n;
        float zbpar=0,zbnep=0;
    float a[100];
    scanf("%d",&n);
    float razlika;
    for(i=1;i<=n;i++)
    {
        scanf("%f",&a[i]);
        if(i%2==0)
            zbpar=zbpar+a[i];
        else
            zbnep=zbnep+a[i];
    }
        razlika=zbpar-zbnep;
    printf("%.2f",razlika);
    return 0;
}
            
        Пример влез
6
2 7.1 12.3 55.9 32 8
Пример излез
24.70
