Да се напише програма коjа за два знаци кои се читаат од стандарден влез (цифри од 0 - 9) ке го отпечати на екран производот на нивните вредности.
#include <stdio.h>
int main() {
char a,b;
int rezultat;
scanf("%c %c", &a, &b);
rezultat = (a%48)*(b%48);
printf("%d", rezultat);
return 0;
}
Пример влез
4 3
Пример излез
12
