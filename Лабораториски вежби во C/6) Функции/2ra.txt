Да се напише функција која ќе го врати производот на сите цифри кои се наоѓаат на парни позиции од даден природен број (гледано од десно кон лево). 
Се смета дека најмалку значајната цифра се наоѓа на 1-та позиција.
#include<stdio.h>
int main()
{
    int br,proizvod=1,pom1,pom2;
    scanf("%d",&br);
    while(br>0)
    {
        pom1=br;
        pom2=(pom1%100)/10;
        if(pom2) proizvod*=pom2;
        br=br/100;
    }
    printf("%d",proizvod);
}
Пример влез
717723
Пример излез
98
