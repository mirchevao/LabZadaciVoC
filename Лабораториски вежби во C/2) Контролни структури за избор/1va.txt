Да се напише програма која за трицифрен број прочитан од стандарден влез ќе провери дали е 
палиндром и ќе испечати порака на екран "Palindrom". Доколку внесениот број не е трицифрен треба да се испечати порака "Ne validen vlez".

#include <stdio.h>

int main(){
    int p, a, b;
    scanf("%d",&p);
    if ((p >= 100)&&(p <=999)){ 
        a = p / 100;
        b = p % 10;
    	if (a == b)
        	printf("Palindrom");
    }
    else
        printf("Ne validen vlez");
    
    
return 0;
}
